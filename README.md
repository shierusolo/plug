# 🔌 Plug

**Plug** es una página de enlaces personales (tipo link-in-bio) con diseño oscuro, minimalista y estético. Ideal para poner en tu bio de Instagram, TikTok, X o donde sea que quieras tener todos tus links en un solo lugar.

[🌐 Ver demo](https://shierusolo.github.io/plug/)

---

## 🎯 Características

- 🎨 Paleta oscura elegante y homogénea
- 📱 Full responsive: se adapta a cualquier pantalla
- ⚡ Rápido, ligero y sin frameworks
- ✏️ Súper fácil de personalizar

---

## 🧰 Cómo usar

1. **Clona el repositorio:**

   ```bash
   git clone https://github.com/shierusolo/plug.git
   cd plug
   ```

2. **Edita `index.html`:**

   Cambia tu nombre, foto y enlaces como quieras:

   ```html
   <h1>Shieru</h1>
   <p>@shierusolo</p>
   <ul>
     <li><a href="https://instagram.com/tuusuario"><i class="icon-ig"></i> Instagram</a></li>
     <li><a href="https://x.com/tuusuario"><i class="icon-x"></i> X (Twitter)</a></li>
     <li><a href="https://tulink.com"><i class="icon-link"></i> Otro link</a></li>
   </ul>
   ```

3. **Súbelo a GitHub Pages** (o tu servidor) y compártelo 🌐

## 🎨 Paleta de colores

Esta es la estética principal de la plantilla:

- `#515151` (gris medio)
- `#404040` (gris oscuro)
- `#2f2f2f` (más oscuro aún)
- `#1f1f1f` (casi negro)
- `#0e0e0e` (negro profundo)

> Perfecto para vibes oscuros, elegantes y modernos 🖤

---

## 🛠 Recomendaciones

- Usa [Font Awesome](https://fontawesome.com/) o SVGs para los íconos
- Puedes modificar `style.css` para cambiar animaciones, fuentes o layout
- También puedes convertir esto en tu mini portafolio o tarjeta digital
