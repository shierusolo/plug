# 🔌 Plug

**Plug** es una pequeña y estética página de enlaces personales (tipo link-in-bio) construida con HTML, CSS y un poco de JavaScript. Es perfecta para compartir todos tus links importantes desde una sola URL, como en tu bio de Instagram, Twitter, etc.

[🔗 Ver demo](https://shierusolo.github.io/plug/)

---

## 🌟 Características

- ✅ Diseño minimalista y responsivo
- ✅ Links con iconos bonitos y animaciones suaves
- ✅ Súper fácil de personalizar: solo editas un HTML
- ✅ Sin frameworks pesados: puro HTML + CSS + JS
